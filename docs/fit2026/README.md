## AI利用の開示方針
学会（ACM/IEEE/Nature等）のコンセンサスに準拠。IPSJが独自ポリシーを出したら要再確認。

- **AIは著者に含めない**（著作権責任を負えないため、全学会共通）
- **Methods**: 「本システムの開発にはAIアシスタント（Claude, Anthropic）を使用した」
- **Acknowledgements**: 「AIアシスタント テディ（Claude）による開発支援・文書作成支援に感謝する」
- **AI査読**: 本文の評価セクション内で既に言及済み（そのまま）

### 根拠
| AI の役割 | 開示場所 |
|-----------|---------|
| コード開発・実験 | Methods（研究手法の一部） |
| 文章推敲・翻訳 | Acknowledgements |
| 査読・レビュー | 本文中で言及 |
