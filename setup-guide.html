Setup Guide

XPathGenie uses Google's Gemini API to analyze HTML and generate XPath expressions. You need a free API key to get started.

1. Get Your Gemini API Key

  1. Go to Google AI Studio → API Keys
  2. Sign in with your Google account
  3. Click "Create API key"
  4. Copy the key (starts with AIza...)
  5. Paste it into the API Key field in XPathGenie

2. Free Tier

Gemini API offers a generous free tier — you can make many requests per day at no cost. XPathGenie typically uses 1–3 API calls per analysis. For most users, the free tier is more than enough.

See Google AI pricing for current limits.

3. Security & Privacy

  • Your API key is sent to the XPathGenie server over HTTPS for Gemini API calls. The server does not log or store your key.
  • You can clear your key at any time by deleting it from the input field.
  • Never share your API key publicly. If compromised, revoke it in Google AI Studio and create a new one.

4. About "Remember API Key" (localStorage)

When you check "Remember API key", your key is saved in your browser's localStorage. This is convenient but comes with trade-offs you should understand:

  • How it works: localStorage is a simple key-value store in your browser, scoped to the site's origin (domain + port). Data persists until explicitly deleted — it survives tab closes, browser restarts, and system reboots.
  • What can access it: Any JavaScript running on the same origin can read localStorage. This includes browser extensions with appropriate permissions and any scripts injected via XSS (cross-site scripting) vulnerabilities.
  • What cannot access it: Other websites, other origins, and server-side code cannot read your localStorage directly.
  • The risk: If a malicious browser extension or an XSS attack targets this site, your stored API key could be exfiltrated. This is a real (though generally low-probability) risk.
  • Mitigation: Gemini API keys can be regenerated instantly and for free. If you suspect your key was compromised, revoke it in Google AI Studio and create a new one.
  • The safe default: Leave "Remember" unchecked. Your key stays in memory only and is cleared when you close the tab. You'll need to paste it again next time, but it's more secure.
